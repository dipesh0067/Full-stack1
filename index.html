About

About This Portfolio

This ePortfolio was created as part of the CET138 Full Stack Development module at the University of Sunderland / ISMT College Butwal. It covers the four core technologies used to build modern websites — HTML for structure, CSS for styling, Bootstrap for responsive layouts, and JavaScript for interactivity.

Each section includes a written explanation of the technology, real working code examples, and a live interactive demo. The portfolio itself is the demonstration — every visual effect, layout, and interaction you see is produced by the code being explained.
